![Java](https://img.shields.io/badge/Java-ff6b6b?style=flat-square&logo=openjdk&logoColor=white)
![Multithreading](https://img.shields.io/badge/Multithreading-4cffee?style=flat-square)

![TLS](https://img.shields.io/badge/TLS%2FSNI-ff88ff?style=flat-square)

> Multithreaded network analysis system built in Java.

- 📦 Parses real **PCAP traffic** files
- 🔁 Tracks **stateful flows** using Five-Tuple hashing
- 🔐 Extracts **TLS SNI** to classify encrypted domains without decryption
- 🚦 Applies **rule-based filtering** with full analytics reporting
- ⚡ Designed for concurrent packet processing with minimal lock contention

### 🛡️ 03 — AudioShield
![React](https://img.shields.io/badge/React-00d8ff?style=flat-square&logo=react&logoColor=white)
![Flask](https://img.shields.io/badge/Flask-a89cf7?style=flat-square&logo=flask&logoColor=white)
![AES-GCM](https://img.shields.io/badge/AES--GCM-ff88ff?style=flat-square)
![SHA256](https://img.shields.io/badge/SHA--256-ff88ff?style=flat-square)
![Status](https://img.shields.io/badge/Status-In%20Development-7c6af7?style=flat-square)

> Passwordless file sharing — your audio file is the key.

- 🎵 Upload any file + an audio file
- 🔑 **SHA-256(audio bytes)** → generates a 256-bit encryption key
- 🛡️ **AES-GCM** encrypts the file with a random 12-byte nonce
- 📦 Receiver gets a **Room Code + QR Code** for access
- 🧹 All files **auto-delete after 24 hours**
- ❌ Even a 1-bit change in audio → decryption fails
